# NLP Preprocessing using NLTK and spaCy

##  Project Description
This project demonstrates text preprocessing using two popular NLP libraries: **NLTK** and **spaCy**.

The goal is to compare their performance in handling basic Natural Language Processing tasks.

---

## Features
- Tokenization
- Lowercasing
- Stopword Removal
- Punctuation Removal
- Stemming (NLTK)
- Lemmatization (NLTK & spaCy)

---

## 📊 Evaluation Metrics
The models are evaluated using:
- Accuracy
- Precision
- Recall
- F1-score

Token-level evaluation is applied for more reliable results.

---

##  Input Sentences
1. I love learning natural language processing.  
2. I do not like waking up early.  
3. Are you going to the party tonight?
